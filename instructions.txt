The password is the following info, in order
a)	total memory of the system (in k)
b)	number of .txt files in “Game” directory (inc. sub directories) that contain the word “kainos”
c)	the processor vendor ID of the machine
d)	the size of “Kainos” folder (in k)
e)	the ubuntu release number of the machine (eg.12.10)
